Term Definition
Network Classifications
LAN Local Area Network - A network that spans a small area, typically within a single building or campus.
MAN Metropolitan Area Network - A network that covers a larger geographical area than a LAN but smaller than a WAN.
WAN Wide Area Network - A network that extends over a large geographical area, often connecting multiple LANs and MANs.
Network Models
Bus Network A network topology in which all devices are connected to a single communication line.
Star Network A network topology where all devices are connected to a central hub or switch.
Protocols
CSMA/CD Ethernet Carrier Sense Multiple Access with Collision Detection - A protocol used in Ethernet networks to manage access to the network medium.
CSMA/CA Wi-Fi Carrier Sense Multiple Access with Collision Avoidance - A protocol used in wireless networks to avoid collisions during data transmission.
Combining Networks
Repeater A network device used to amplify or regenerate signals in order to extend the reach of a network.
Bridge A network device that connects two or more network segments and forwards traffic between them.
Switch A network device that uses MAC addresses to forward data only to the intended recipient.
Router A network device that forwards data packets between computer networks, typically used in WANs.
Gateway A network node that connects different networks using different protocols.
Client/Server Model
P2P (Peer-to-Peer) A decentralized model where each node can act as both a client and a server.
Distributed System A network where tasks are divided between multiple computers, often working in parallel.
Cloud Computing A model that enables on-demand access to a shared pool of computing resources over the internet.
Internet Addressing
Tier 1 Top-level ISPs that have direct access to the global internet backbone.
Tier 2 Regional ISPs that connect to Tier 1 ISPs and smaller ISPs.
Tier 3 Local ISPs that connect to higher-tier ISPs to access the internet.
IP Address
ICANN Internet Corporation for Assigned Names and Numbers - Responsible for coordinating IP address allocation and DNS root zone management.
DNS Domain Name System - Translates domain names into IP addresses.
TLD (Top-level Domain) The highest level in the hierarchical domain name system (.com, .org, .net, etc.).
Country-code TLDs TLDs assigned to specific countries or territories (e.g., .us, .uk, .fr).
Internet Protocols
FTP File Transfer Protocol - Used for transferring files between a client and a server on a computer network.
HTTP Hypertext Transfer Protocol - The protocol used for transmitting web pages over the internet.
SMTP Simple Mail Transfer Protocol - Used for sending emails between servers.
World Wide Web
URL Uniform Resource Locator - A web address that specifies the location of a resource on the internet.
Client-side Activities Activities performed by the user's web browser on their device.
Server-side Activities Activities performed on the web server that hosts the website.
Internet Four-layer Software
Application Layer Provides interface for users and applications to access network services.
Transport Layer Responsible for end-to-end communication and error-checking.
Network Layer Manages data routing, addressing, and forwarding.
Link Layer Deals with data transfer between nodes on the same network segment.
Internet Characteristics
Packet-switching Network Data transmission method where information is divided into packets for efficient routing.
Fault Tolerant Capable of continuing operation even in the presence of faults or errors.
Redundant Having additional elements for backup or alternate functioning.
Cybersecurity
Malware Malicious software designed to disrupt, damage, or gain unauthorized access to computer systems.
Virus A type of malware that self-replicates and spreads by inserting copies of itself into other programs or documents.
Worm A standalone malware computer program that replicates itself to spread to other computers.
Trojan Horse A type of malware disguised as legitimate software to trick users into installing it.
Spyware Software that collects information about a user's activities without their knowledge.
Phishing Attempt to acquire sensitive information by masquerading as a trustworthy entity in electronic communication.
DDoS Attack Distributed Denial-of-Service Attack - An attempt to make an online service unavailable by overwhelming it with traffic from multiple sources.
Protection
Firewall A security system that monitors and controls incoming and outgoing network traffic.
Spam Filter Software used to detect and filter out unsolicited and unwanted emails.
Proxy Server A server that acts as an intermediary for requests from clients seeking resources from other servers.
Antivirus Software Software designed to detect, prevent, and remove malicious software.
Cryptography The practice and study of secure communication techniques.
Public Key Encryption An encryption system that uses two keys: a public key and a private key, for secure communication.
